# 📊 Natural Gas Price Analysis (JP Morgan Virtual Experience)

## 📌 Overview

A data analysis project based on a virtual experience program, focused on analyzing and forecasting natural gas prices.

The project involves working with historical price data to identify trends and build basic predictive insights relevant to the energy market.

---

## ⚙️ Tech Stack

- Python  
- Pandas  
- NumPy  
- Matplotlib  

---

## ✨ Features

- Processed and cleaned historical natural gas price data  
- Performed time-series trend analysis  
- Visualized price patterns using graphs  
- Built basic forecasting logic for future price estimation  

---

## 🧠 What I Implemented

- Data cleaning and preprocessing using Pandas  
- Time-series analysis to identify seasonal trends  
- Visualization of price fluctuations  
- Implemented logic for price estimation based on historical patterns  

---

## ▶️ How to Run

1. Clone the repository  
2. Install dependencies:
   ```bash
   pip install pandas numpy matplotlib
   ```
3. Run the Python script:
   ```bash
   python main.py
   ```
